# Moonveil - Ultimate minecraft site
- This is the frontend interface for Modern site Minecraft. Built with React, Vite, and Tailwind CSS.


# Prerequisites
Before you begin, ensure you have met the following requirements:
- Node.js: v18.x or higher.
- NPM 

# Installation Guide
- Open your terminal and navigate to this frontend folder.
- Install the required dependencies:

```bash
npm install
```

# Running in Development
- To start the development server with hot-reload:

```bash
npm run dev
```
- Open your browser (or the port shown in terminal).


# Building for Production
- To deploy this frontend, you need to build it into static files (HTML/CSS/JS).

- Run the build command:
```bash
npm run build
```
- The output files will be in the dist/ folder.
- Upload the contents of the dist/ folder to your web hosting (Vercel, Netlify).
